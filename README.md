# League Simulation

4-team football league simulation. Go backend + Vue.js frontend.

## Tech Stack

- **Backend:** Go, Gin, GORM, JWT, Swagger
- **Frontend:** Vue 3, Pinia, Chart.js, Tailwind
- **Database:** PostgreSQL (Supabase)
- **Cache:** Redis (Upstash)
- **Deploy:** Render (backend) + Vercel (frontend)

## Setup

```bash
git clone https://github.com/yahyayesilyurt/league-simulation.git
cd league-simulation
cp .env.example .env
docker-compose up
```
